# 《我带了一整个地球穿越大唐》剧情纲总控手册（主文件）

## 1. 使用定位

本文件是 04_剧情大纲 的唯一总控入口，用于统合《故事背景大纲》、人物卡、分卷路线与章节扩写方向。
后续所有卷纲、章纲与正文，均以本手册的主轴、卖点和升级节奏为最高口径。

## 2. 总体目标（长篇规划）

- **总字数**：约 200 万字以上
- **总卷数**：约 8 卷
- **单卷建议**：20 万至 30 万字，60 至 80 章
- **总控节奏**：生活流起手 + 高门暧昧牵引 + 工业化滚雪球 + 皇室与五姓七望争抢主角 + 对外扩张吃资源 + 文明跃迁爽感
- **总控原则**：弱内斗、强外扩、强工业化、强情绪满足，蓝星只做幕后引擎，不抢大唐前台主戏

## 3. 主线命题与终局

### 主线命题
现代青年房兵在高门深宅中醒来，先靠吃穿住行和治病救命稳住房家与皇室，再靠商业、工业、科学院、卫生体系和军工体系不断放大生产力，最终让皇室与五姓七望都不得不争着把女儿、资源与政治信用押到他身上，一起推动大唐完成由内而外的帝国扩张。

### 三重矛盾
1. **文明跃迁 vs 旧秩序惯性**：门阀礼法、旧式官僚和落后生产力，会持续阻碍主角推进工业化、标准化和公共治理。
2. **个人快活 vs 多方绑定**：主角本想做个有钱有势有美人的高门浪子，却越做越大，被皇室、公主、门阀贵女和天下资源一起推上牌桌中央。
3. **扩张胃口 vs 帝国承载**：对外扩张能带来劳力、矿产、航路与财富，但也会逼迫大唐同步升级交通、军工、卫生、教育与组织能力。

### 终局交付
- **帝国升级**：大唐提前完成早期工业化、公共卫生制度化、科学院成型与全球航路布局。
- **个人登顶**：房兵摆脱历史中的被动婚配宿命，成为皇室离不开、门阀抢着嫁、军政商学都绕着转的无冕之王。
- **双线闭环**：蓝星治理经验和物资权限彻底沉入幕后，成为主角驱动大唐文明跃迁的稳定底盘。

## 4. 全书核心卖点

### 1. 高门求婿与暧昧围猎
- 五姓七望前期把主角当异数和风险，中期开始派嫡女试探、合作、暧昧、联姻，后期干脆把婚约与产业一起打包押注。
- 高阳、长乐与门阀贵女不只是点缀，她们既提供长期情绪价值，也承担皇室绑定、门阀拉拢与主角上位的功能。

### 2. 事业流不是单纯赚钱
- 从酒楼、白酒、制糖、成衣，到钱庄、基建、铁路、军工、能源、航海，每次升级都必须带来新秩序、新阶层和新版图。
- 产业推进必须与科学院、工匠体系、标准化和卫生普及同时升级，形成“文明被一层层点亮”的满足感。

### 3. 对外扩张才是长篇主战场
- 中前期只保留足够的高门压迫感、朝堂阻力和权力试探，不把储位斗争写成长期主戏。
- 真正的大冲突放在草原、西域、矿区、航路、海港与海外资源点，把扩张、征服、掠夺和建设写成卷级高潮。

### 4. 蓝星只做幕后反哺
- 蓝星继续保留治理、回馈、具现的硬逻辑，但其存在感只服务于解决瓶颈、释放权限、补足生产资料和管理范式。
- 夜线负责给答案、给压力、给资源边界，白天的大唐现实负责情绪爆发、人物关系和历史改写。

### 5. 生活流必须一直在场
- 再大的制度、工业和扩张，也要从一桌菜、一次治病、一件新衣、一段推拉、一场宴饮、一条铁路样轨落地。
- 主角的魅力来自现代人的松弛、分寸、平等感与执行力，而不是空喊口号。

## 6. 分卷执行原则

### 卷级必须同时回答的四件事
1. 本卷主角通过什么生活流切口把大事做成？
2. 本卷工业、卫生、科学院、军工里至少推进哪两条长期主线？
3. 本卷由哪一股外部力量提供最大冲突与最大资源回报？
4. 本卷哪位公主或高门贵女完成了新的情感推进、合作推进或婚议推进？

### 冲突设计硬规则
- 内部政治只保留门阀施压、老臣怀疑、皇室试探和局部储位阴影，不再长期占据主戏。
- 每卷都要有可见的产业跃迁、组织升级和版图外扩，不能只写朝堂争执。
- 每卷都要有让读者感到“又有人想把女儿嫁给他”“又有人想跟着他吃更大的肉”的情绪回报。
- 每卷都要让敌人输在旧秩序、旧认知、旧效率，而不是靠空泛嘴炮。

## 7. 长期主线挂载

### 五姓七望线
- 前期：试探、观察、轻视、暗中评估。
- 中期：派贵女接近、给产业合作、试婚约口风、争夺站队名额。
- 后期：共同投资铁路、矿山、港口、科学院和海外商路，门阀从压力源变成扩张合伙人。

### 科学院线
- 卷三建立雏形，卷四正式立名，卷五参与边地工程，卷六成为能源与军工中枢，卷七服务海洋扩张，卷八沉淀为帝国制度成果。

### 卫生普及线
- 卷一以急救药救命立威，卷二推进洗手、煮水、通风和简易医护常识，卷四形成训练体系，卷六覆盖军队与矿区，卷七进入港口与航海体系，卷八成为帝国公共治理的一部分。

### 工业化线
- 商业爆款起盘 -> 农政与基建稳盘 -> 煤铁蒸汽点火 -> 标准化与科学院扩容 -> 铁路火器外扩 -> 石油机械跃迁 -> 海洋工业成型 -> 帝国工业底盘定型。

### 感情线
- 高阳负责锋利、主动与占有欲。
- 长乐负责温柔、信任与皇家绑定。
- 门阀贵女负责评估、试探、合作、婚议与门第抬升。
- 双铃、花魁等人继续提供烟火气与亲密感，但高门暧昧必须承担卷级推进功能。

## 8. 阶段性检查清单（卷到章）

### 卷前三问
1. 本卷的最大爽点，是产业起飞、扩张得手，还是高门联姻落地？
2. 本卷主角到底把哪一拨人变成了自己人，是皇室、门阀、工匠、军方还是边地势力？
3. 本卷结束时，大唐相较开卷具体多了什么，铁路、学院、卫生、军队、港口还是海外据点？

### 章级检查清单
- [ ] 这一章是否有生活流烟火气，不只是谈事？
- [ ] 这一章是否把工业、卫生、扩张或感情线中的至少一条往前推了一步？
- [ ] 这一章的阻力是否来自真实旧秩序，而不是空泛争吵？
- [ ] 这一章是否留下了下一章的钩子，大人物震动、贵女失衡、工程突破或扩张收益？
