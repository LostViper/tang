# 《我带了一整个地球穿越大唐》剧情纲总控手册（主文件）

## 1. 使用定位

本文件是 04_剧情大纲 的唯一总控入口，用于统合《故事背景大纲》与各卷的剧情推进路线。
所有后续分卷大纲与章节小纲的创作，必须先对齐本文件的宏观规划。

## 2. 总体目标（长篇规划）

- **总字数**：约 200万字以上
- **总卷数**：约 8-10 卷
- **单卷建议**：20万-30万字（约 60-80 章）
- **核心节奏**：《唐砖》式的生活流起手 + 降维打击滚雪球 + 极具压迫感的朝堂/天下风暴。

## 3. 主线命题与终局

### 主线命题
现代青年房兵（房遗爱）在“大唐现实”与“蓝星意识”的双重统治中，用降维的生产力与制度逐步改造大唐，同时完成自己对权势、财富、美人与历史命运的全面收割。

### 三重矛盾
1. **历史惯性 vs 现代降维**：大唐旧有的门阀、农业、皇权结构，与主角带来的工业化、新资本、新农政的剧烈摩擦。
2. **个人享乐 vs 天下重担**：主角本想当个快活的高门衙内，却随着摊子越铺越大，被历史的进程推着往前走，不得不承担起大唐国运与蓝星治理的双重重压。
3. **蓝星绝对威权 vs 大唐现实博弈**：夜里蓝星的“一言九鼎”与白天大唐的“人心难测、效率低下”形成强烈反差，逼迫主角在两个世界间切换统治面具。

### 终局交付
- **大唐重塑**：大唐提前完成初步工业化与全球化地理开拓。
- **个人登顶**：房遗爱跳出“历史悲剧驸马”的宿命，成为大唐事实上的幕后无冕之王。
- **双星闭环**：大唐与蓝星的资源与治理经验彻底融会贯通，主角成为跨越时空的真正统治者。

## 5. 核心特色与叙事手法 (强制要求)

### 1. 《唐砖》式的生活流起手
- **禁止一上来就讲大道理/大朝政**。
- 所有重大政治与经济变革，必须从**一顿饭、一个玩具、一把椅子、一次治病**这种极具烟火气的生活细节切入。
- **爽点机制**：用现代日常生活的降维产物，对大唐权贵进行“降维装逼”，在轻松诙谐的日常互动中，不经意间抛出能改变国本的炸弹（如木制自行车引出物理与机械，糖醋鱼引出制糖业）。

### 2. 皇家合伙人模式
- 主角不能走传统的“臣子死谏”或“苦逼打工”路线。
- 必须通过**利益捆绑**（如把暴利工坊的干股送进皇后内库），把皇帝、皇后、公主变成自己的“大股东”和“保护伞”。
- 让反派在弹劾主角时，屡屡踢到皇室护短的铁板，形成强烈的反差爽感。

### 3. 皇子群像与夺嫡暗流
- **李承乾**：因为主角的出现，他得到了最强的外援，但也承受着改变历史轨迹的压力。
- **李泰**：被主角的“奇技淫巧”吸引，带着学术与夺嫡的双重目的接近。
- **李恪**：欣赏主角的江湖气与烈酒，试图拉拢。
- 主角的态度：**游刃有余，和所有人做生意，但不做任何人的狗。**

### 4. 蓝星双线反哺机制
- 蓝星绝不是一个只会吐物资的静止仓库。
- 主角白天在大唐遇到瓶颈（如物流不畅、工匠管理混乱），夜里就在蓝星处理全球供应链危机，醒来后将**现代管理制度、流水线、仓储模型**降维应用到大唐工坊中。

## 6. 阶段性执行规范 (卷到章)

### 卷前三问
1. 本卷要抛出哪几个核心“降维产物”（农业、工业、医疗）？
2. 本卷主角如何通过这些产物，把哪个层级的大佬（皇帝/重臣/门阀）拉下水？
3. 本卷谁来当反派？反派如何因为眼红而自寻死路（反派闭环）？

### 章级检查清单
- [ ] 这一章有没有“生活流”的烟火气细节？
- [ ] 主角的话有没有太像“机械AI”或“空洞装逼”？（必须保持真实人气和适度疲惫/自嘲）
- [ ] 抛出的现代知识是否考虑了“大唐现有的生产力边界”？（不能一步登天造航母）
- [ ] 章末是否留下了让读者期待下一章的“钩子”（悬念/反转/大人物的震惊）？
