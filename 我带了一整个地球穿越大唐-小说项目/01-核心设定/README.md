# 01-核心设定

本目录专门整理“现实中只有贞观时空，现代青年房兵魂穿房遗爱，并以意识地球 / 意识蓝星为接口切入其中”这一核心前提下的专项材料框架。当前版本不再采用“现代地球复制体整体进入贞观”的设定，而是把主角意识中那颗停留在 2026 年科技水平的高保真意识地球，作为知识、推演、组织统摄与少量物质具现的唯一超常来源。这里不直接替代世界观、剧情和人物文档，而是为后续所有推演提供底层约束、问题清单与扩写入口。

## 使用方式

- 先阅读“核心设定总纲”，统一底层假设与叙事边界。
- 阅读时默认现实世界只有大唐，不再存在一个同步落地的现代全球系统。
- 把“主角意识中的意识地球”理解为认知与组织接口，而不是可直接降临现实的第二个世界，避免把认知外挂误写成现实无限资源。
- 再按专题补充资料，避免不同维度各写各的导致设定冲突。
- 每个专题都保留“关键问题”“直接影响”“可扩写条目”三类内容，方便后续转化成剧情、时间线或章节素材。

## 文件结构

- `00-核心设定总纲.md`：统一说明单现实世界前提、意识地球接口、具现规则、初始状态与总限制。
- `01-地理与空间格局.md`：处理贞观现实地理与主角如何借意识地球提升空间认知。
- `02-人口与制度接驳.md`：处理大唐人口与制度如何被主角带来的现代治理认知持续冲击。
- `03-物资储备与工业底盘.md`：处理主角如何用少量具现去撬动贞观现实中的工坊、药材、运输和生产能力。
- `04-技术传播与能力边界.md`：处理意识地球如何提供知识与样品，以及贞观现实如何学习、试错、复做和扩散。
- `05-政治重构与权力冲击.md`：处理主角在大唐权力结构中被理解、利用、防范或神化的过程。
- `06-文化碰撞与认知断层.md`：处理主角带来的现代认知、价值观和奇迹化行为与贞观社会之间的深层冲突。
- `07-社会秩序与连锁冲击.md`：处理主角影响现实后引发的治安、舆情、迁徙、公共卫生与阶层震荡。
- `08-五姓七望与门阀势力.md`：处理顶级士族集团的家族构成、权力来源、婚姻秩序，以及主角与门阀之间的长期博弈。
- `09-初唐饮食习惯.md`：处理武德至贞观时期的主食、副食、饮品、烹饪方式、阶层差异、地域差异与礼法禁忌。
- `10-贞观长安坊市寺观地标.md`：处理长安核心坊区、寺观、东市西市与朱雀街东西两侧的空间功能和历史节点。
- `11-贞观元年朝堂人物群像.md`：当前已按贞观四年的架空时间锚点处理皇室、宰相重臣、军方、宗室与其他关键人物的即时状态、关系线与剧情入口。
- `12-贞观年间炼钢水平.md`：处理唐初生铁、炒钢、百炼钢、灌钢、官营冶铸、军器材料与技术上限。
- `13-贞观年间制盐水平.md`：处理唐初海盐、池盐、井盐、岩盐、盐政、长安供盐与制盐工艺边界。
- `14-贞观年间农耕水平.md`：处理均田制、租庸调、主要作物、农具、水利、亩产与关中粮食底盘。
- `15-贞观年间交通情况.md`：处理驿道、漕运、丝路、海路、关津制度与长安交通中枢结构。
- `16-贞观年间朝廷财政收入.md`：处理租庸调主体、布帛与粮食财政、次要税源与中央支出逻辑。

## 目标

- 让核心设定拥有独立资料区，便于长期维护。
- 让每个专题都能直接支撑后续情节推演。
- 让宏观设定与微观剧情在同一套约束下展开。
