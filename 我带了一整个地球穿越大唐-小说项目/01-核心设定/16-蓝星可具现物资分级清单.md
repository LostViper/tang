# 蓝星可具现物资分级清单

## 核心定位

- 这份清单不是商城目录。
- 它只是把蓝星当前理论上可具现、但必须服从现实边界的物资，整理成一份剧情可用分级表。
- 主角能不能把某样东西带出来，首先看的是具现周期、体量上限、对象边界和大唐现实承接能力，而不是额外点数。

## 现行硬边界

- 每七天一次稳定具现
- 单周期总量原则上不超过 0.2 立方米或 50 公斤
- 仅限非生命、性质稳定、边界清晰对象
- 具现物只能承担样品、补丁、备件、救急、试手角色
- 不能长期代供，不能顶替现实工坊、物流与组织能力

## 判断顺序

每次决定具现前，都至少先问四件事：

1. 这一周期值不值得用  
2. 单次体量是否装得下  
3. 大唐现实接不接得住  
4. 来路能不能讲圆、后果能不能收住  

凡是第四条过不去的，再好也先不具现。

## 物资分级总表

### A 级：前期低风险剧情物资

**定位**

- 适合日常剧情推进
- 适合试口、试人、试家内接受度
- 适合商业试手与关系润滑

**典型物资**

- 去包装奶糖
- 辣味零嘴、面筋条、果干
- 小批量调味料样品
- 小批量酒样
- 无现代标识的食材样品
- 适合二次装瓶、二次盛放的小样物资

**适用场景**

- 房家日常互动
- 酒楼试菜、试酒
- 同辈圈试口与拉近关系
- 小规模生意试手

**限制**

- 不能长期开仓供货
- 必须先去包装、去标识、重装容器
- 更适合做“新鲜感”和“第一口记忆点”，不适合直接当主供应链

### B 级：民生与医药救急物资

**定位**

- 用于救命、抢时、补脆弱节点
- 最能快速建立信用，但也最容易引来过度关注

**典型物资**

- 常见成药
- 基础退热、止痛、止泻类药片
- 基础急救耗材
- 消毒用品
- 标准化试剂
- 小批量医用辅材

**适用场景**

- 救急病患
- 药房试验
- 关键人物保命
- 医药线点火

**限制**

- 只适合急救和样品验证
- 不能把现代药品写成源源不断的长期供给
- 每次使用都要考虑解释成本和被追索风险

### C 级：工艺点火物资

**定位**

- 用于跨过技术门槛
- 用于让某条现实路线先跑通第一步

**典型物资**

- 标准件样品
- 精密备件
- 密封件
- 少量高纯材料
- 工艺验证样件
- 配方样品
- 小批量实验工具

**适用场景**

- 工坊试制
- 冶炼、盐业、酒业流程验证
- 军器作坊关键补件
- 商业生产的第一轮打样

**限制**

- 只能点火，不能代工
- 重点是让唐人复做出来，不是一直靠主角补件
- 一旦现实体系无法跟上，这类具现价值会迅速下降

### D 级：高敏感战略物资

**定位**

- 关系到军政、工业、长期秩序优势
- 价值高，风险也极高

**典型物资**

- 军工关键件
- 高精度控制板卡
- 特殊高纯材料
- 复杂实验样机中的关键单元
- 对战局或大型工艺路线有决定作用的小体积核心件

**适用场景**

- 军政生死节点
- 大规模变局前的关键校验
- 某条高价值路线的第一次定标

**限制**

- 需要明确现实承接路径
- 不适合前期频繁动用
- 一旦暴露，极易改变各方对主角的政治判断

### X 级：禁止或长期冻结项

**定位**

- 原则上不进入常规具现池
- 即使理论上存在，也不应用于前中期剧情落地

**典型对象**

- 活体
- 正在运行的系统
- 持续输出的能量
- 大型成套设备
- 整条工业产线
- 大宗粮食与燃料
- 大唐现实根本无力维护的复杂工程对象

**原因**

- 这些对象会直接打穿现有边界设定
- 即使短期出现，也只会把剧情推向“神迹展示”，而不是“现实承接”

## 前期剧情优先物资池

### 日常线

- 去包装奶糖
- 辣味零嘴
- 果干
- 小批量特色调味料

### 酒楼线

- 分层酒样
- 可重装进越州青瓷瓶的高端酒体样品
- 适合试炒的菜品样料
- 后厨试味用调料样品

### 民生线

- 基础药片
- 敷料
- 基础消毒耗材

### 工艺线

- 小批量标准件
- 配方验证样品
- 少量精密工具

## 使用顺序建议

- 前期先用 A 级和 B 级建立现实信用与日常触感
- 中期逐步把 C 级投向最值得点火的工艺断点
- D 级只在真正会改变大盘结果的时刻动用
- X 级长期封存，只作为边界提醒存在

## 叙事使用原则

- 任何一次具现，都优先写“为什么值得具现”，再写“带出了什么”
- 任何一个物资，都优先写它如何嵌进现实人、事、工坊、商路，而不是单独列数据
- 任何高价值物资，都必须附带：
  - 来路遮掩方案
  - 现实承接人
  - 一旦失败的止损想法

## 一句话总结

- 冷却与体量决定主角这次能带出多少
- 对象边界决定主角什么根本不能带
- 大唐现实决定这些物资最后能不能变成真正的优势
