# 优化第15章计划

## 1. 目标与背景 (Summary)
根据最新整合的 `章节正文文风规范与前置检查.md`，对 `11-章节草稿/16-第十五章-筹备开业.md` 进行彻底的文风清洗与重写。该章原稿存在极度严重的“单行碎句”排版，且充斥着“压场面”、“把场子带热”、“把风声吹到”等现代口水话和被明令禁止的AI高频词（如“压、定、拉、爆、炸”）。本次优化旨在将碎片化的句子合并为顺畅紧凑的自然段，去除现代口语，保留大唐勋贵子弟的身份感，并突出主角利用现代商业模式（股份制、饥饿营销、外卖、VIP会员）对大唐餐饮业进行“高级降维打击”的爽感。

## 2. 当前状态分析 (Current State Analysis)
- **排版问题**：全篇几乎都是单句换行，视觉极其碎片化，破坏了阅读连贯性，不符合“紧凑顺读”的规范。
- **违规词泛滥**：大量使用了“压给市井”、“压场面”、“带热”、“拉了一栏”、“炸了一层”、“定死”等违规或现代口水化词汇。
- **对话出戏**：程处弼、长孙涣等人的对话有时像现代职场混混，缺乏国公府公子的基本分寸与唐代语境。高阳公主的出场也稍显刻意。

## 3. 拟进行的修改 (Proposed Changes)
- **段落合并**：将单行对话与动作描写合并为完整的自然段。做到先说清动作和判断，再引出对话。
- **违规词清洗**：
  - “压给市井” -> “投入市井”
  - “拉了一栏” -> “单列了一项”
  - “压场面” -> “镇住局面”
  - “带热” -> “挑起兴致”
  - “炸了一层” -> “引起轩然大波”
  - “定死” -> “立稳”
- **对话去口水化**：重写主角分股份、立规矩的对话，用更贴合唐代商业与勋贵圈的词汇（如“入股”、“红利”、“规矩”、“镇场”）。
- **强化降维打击感**：在描写高背椅、铁锅炒菜、分档售酒、外送服务时，突出这些现代商业模式对唐人认知的冲击，让爽感“自然流露”。

## 4. 假设与决策 (Assumptions & Decisions)
- **保留核心情节**：分股份、门前悬赏对联、高背椅与铁锅炒菜的引入、平康坊花魁的试探、高阳公主的关注等核心情节全部保留，仅修改呈现方式。
- **高阳公主的合理化**：高阳的关注不再是单纯的“宫人八卦”，而是通过相府送进宫的某种新奇玩意或确实轰动全长安的诗词引出，使其好奇心更加合理。

## 5. 验证步骤 (Verification Steps)
1. 全文搜索确认不存在“压、盘、线、定、拉、爆、炸”等违规AI词汇（除非作为普通动词的合理使用，如“按压”、“盘子”，但此处为避免误判，尽量全清）。
2. 确认单行碎句已被全部合并为逻辑连贯的自然段。
3. 检查人物对话是否去除了现代职场/网文口水话，符合唐代语境。
