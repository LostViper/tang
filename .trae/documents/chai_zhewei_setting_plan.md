# 柴哲威推主角落水动机设定优化计划

## 一、 当前状态分析
目前在大纲和正文草稿中：
1. **落水事件**：第一章明确了主角房遗爱穿越前，原身是被人推下水的，房玄龄正在暗中追查。
2. **柴哲威定位**：在 `06-反派二代人物卡.md` 中，柴哲威被设定为“轻浮挑衅型”，是早期当众找事的反派，看不起房遗爱。
3. **高阳公主**：在 `03-高阳公主人物卡.md` 中，她是李世民之女，原定要许配给房遗爱。

**缺失的逻辑链**：目前并没有明确点出“究竟是谁推了房遗爱落水”，以及“为什么要推他”。

## 二、 优化目标（用户需求）
新增核心设定：**柴哲威对高阳公主倾心，但在权贵圈中收到风声，得知李世民有意将高阳许配给房遗爱。出于嫉妒和不忿，柴哲威暗中推房遗爱下水。**

这个设定的好处：
1. **动机扎实**：将柴哲威从一个“为了反派而反派”的脸谱化人物，变成了有明确情感和利益动机（抢夺公主婚配资源）的立体反派。
2. **冲突前置**：让男主一穿越过来，就与反派阵营（柴家等勋贵）结下了不仅是政治上、更是夺妻之恨的死仇，极大增强了前期同辈交锋的宿命感和火药味。
3. **完美闭环**：解释了第一章落水事件的真凶，也为后续朝堂上柴家疯狂针对房家找到了更深层的私怨逻辑。

## 三、 具体修改方案
需要修改两处核心设定文件：

### 1. 修改 `03-人物/06-反派二代人物卡.md`
在“柴哲威”的词条下，补充其对高阳的倾慕以及推人落水的核心前史。

**新增内容**：
```markdown
### 核心前史（推人落水真凶）
- 柴哲威一直对明艳火辣的高阳公主倾心，并自诩以柴家的门第足以匹配公主。
- 但他在权贵圈中收到暗中风声，得知李世民竟有意将高阳许配给名声不堪的房遗爱。
- 出于极度的嫉妒与不忿，柴哲威在一次同辈聚会中，趁乱暗下黑手，将房遗爱推入水中，意图毁掉这桩婚事，这也直接导致了现代男主房兵的穿越。
- 这份“夺妻之恨”加“杀身之仇”，让柴哲威成为男主前期必须彻底踩死的死敌。
```

### 2. 修改 `04-剧情大纲/01-第一卷大纲.md`
在第一卷大纲的背景或反派设定部分，将这起“落水谋杀案”的真相和动机明确记录下来，作为后续揭秘的伏笔。

**新增内容**（在大纲的“核心反派与冲突线”或“开篇悬念”处）：
```markdown
* **落水悬念的真相**：开篇房遗爱落水并非意外，真凶是柴绍之子柴哲威。柴哲威倾心高阳公主，听闻皇室有意将高阳下嫁房家，出于嫉妒暗下黑手。这一设定将同辈圈的争斗与皇室婚配资源直接挂钩，为后续房兵在朝堂和商场上彻底干翻柴家埋下不死不休的死仇伏笔。
```
