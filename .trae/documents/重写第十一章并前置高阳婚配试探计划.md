# 重写第十一章并前置高阳婚配试探计划

## Summary

本次目标不是局部润句，而是**重写第十一章**，把原本以“临水阁翻盘后的三层分流”为主的章节，改成一章**宫中线占主位**的新版本：

- 正面安排李世民、长孙皇后、高阳公主出场
- 明确写出李世民准备安排房遗爱的婚事
- 高阳起初对“那个憨傻的房遗爱”本能排斥
- 随后听见落水、开窍、临水阁两首诗的风声，转为起疑
- 最终形成“高阳准备亲自去试探一下房遗爱”的章节落点

根据本轮确认范围，这次执行同步到：

- 第十一章正文
- 第一卷章节小纲中第十一章对应部分

不扩展修改其他章节正文，但计划会保证与第七章“婚事已被李世民私下提过”的新设定、以及第十章“风吹回父辈耳中”的现有方向保持一致。

### 5. 当前最适合的改法是“保留风声来源，重写章节重心”

结合用户偏好“宫中线占主位”，最合适的方向不是简单给现有第十一章末尾多加一段宫中戏，而是：

- 保留临水阁翻盘余波作为消息来源
- 但把第十一章真正的重心挪到宫中
- 让这一章承担“皇室第一次正式把房遗爱拉进婚配盘面”的功能

也就是说，第十一章不再主要负责“同辈分流”，而要改成：

- 风声入宫
- 李世民与长孙皇后谈婚配价值
- 高阳先抗拒、后起疑
- 决定亲自去试探房遗爱

这样一来，章节吸引力会更强，且能把卷一后段公主线提前点燃。

## Proposed Changes

### 1. 重写第十一章结构，改为“宫中线主导”的双层推进

- **拟修改文件**：
  - `/Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/11-章节草稿/12-第十一章-谁会站过来，谁会记下仇.md`
- **怎么改**：
  - 放弃当前以同辈分流为主体的章内结构
  - 改成更紧凑的四段式：
    1. 临水阁风声快速扩散，作为消息源入宫
    2. 李世民与长孙皇后先谈房遗爱婚配价值与人选问题
    3. 高阳出场，先因“憨傻房遗爱”而强烈不愿
    4. 听见落水开窍与临水阁两首诗后起疑，决定亲自试探
- **为什么改**：
  - 用户已明确要求“宫中线占主位”
  - 只追加宫中段落，无法真正实现章节重心切换

### 2. 让李世民与长孙皇后的对话承担“婚配盘面正式开桌”的功能

- **拟修改文件**：
  - 第十一章正文
- **怎么改**：
  - 李世民的功能重点：
    - 点出房遗爱原本不是理想婚配对象
    - 但近期风声起得太快，已不能再按旧印象看
    - 婚配不只是儿女事，而是房家、皇室、朝局资源重新归位的问题
  - 长孙皇后的功能重点：
    - 比李世民更早看见“婚配价值”
    - 不急着定人，却会更敏锐地看出高阳与房遗爱这条线的危险性与可控性
    - 她既懂皇家分寸，也懂女儿脾气，能把这场婚配谈得更柔，却更深
- **为什么改**：
  - 这能让宫中戏不是纯信息投放，而是真正带出皇室视角

### 3. 把高阳的情绪曲线写成“先厌、后疑、再主动”

- **拟修改文件**：
  - 第十一章正文
- **怎么改**：
  - 第一层：
    - 高阳一听“房遗爱”就先本能排斥
    - 因为她认知里的房遗爱仍是那个憨傻、好笑、扶不起来的名门废物
  - 第二层：
    - 她又听见落水、变了个人、临水阁两首诗
    - 尤其“会写诗”和“敢当场压局”这两点，会让她产生明显反差感
  - 第三层：
    - 她不信别人嘴里的夸张风声
    - 也不愿直接接受父皇安排
    - 于是决定自己去看、自己去试
- **为什么改**：
  - 用户要的不是高阳立刻动心
  - 而是“起了好奇心，准备去试探”
  - 这条情绪线必须写得有台阶，才不突兀

### 4. 把“临水阁两首诗”写成高阳好奇心的直接引信

- **拟修改文件**：
  - 第十一章正文
- **怎么改**：
  - 不只笼统说“风声传入宫”
  - 要让宫中明确听见：
    - 房遗爱落水之后变了
    - 在临水阁压住了场
    - 两首诗已经被人传诵
  - 高阳的反应不应只是“听见了诗”
  - 而应是：
    - 她一开始觉得荒唐
    - 再觉得这不像旧房遗爱会做出来的事
    - 最后因为“不像”而生出试探欲
- **为什么改**：
  - 用户点名“两首诗”
  - 说明这不是背景噪音，而是高阳情绪转向的硬触发器

### 5. 保留少量“风从临水阁吹起”的地面视角，但不再让其压过宫中戏

- **拟修改文件**：
  - 第十一章正文
- **怎么改**：
  - 可以保留很少量的开头或过桥段：
    - 长安在传
    - 相府在听
    - 风一路吹进宫里
  - 但不再展开原版那种大篇幅同辈分流
  - 原来属于第十一章的“谁站过来、谁观望、谁记仇”，若需要，只保留成宫中谈资里的一句结果性判断
- **为什么改**：
  - 要给宫中线让出主位
  - 同时保住上下文来源，不至于像凭空切景

### 6. 同步更新第十一章小纲，使其改成“婚配风声正式入局章”

- **拟修改文件**：
  - `/Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/04-剧情大纲/13-第一卷章节小纲.md`
- **怎么改**：
  - 把第十一章的剧情目标从“分流”改成：
    - 临水阁风声吹入皇室
    - 李世民与长孙皇后把房遗爱重新放进婚配盘面
    - 高阳由嫌恶转为好奇并决定试探
  - 章末钩子应改成：
    - 不是单纯“谁会站过来、谁会记下仇”
    - 而是“高阳已经被勾起心思，准备亲自去碰一碰这个新房遗爱”
- **为什么改**：
  - 这样后续再写公主线时，口径会更稳
  - 也避免当前小纲与正文功能冲突

### 7. 严控文风与信息表达方式

- **拟处理原则**：
  - 行文保持轻松、顺口、有趣味
  - 但不写成纯宫斗说明书
  - 不直接出现“第十章”“第十一章”之类元叙述
  - 不让李世民、长孙皇后说话像设定介绍员
- **怎么落笔**：
  - 用人物对话、神态、偏好与判断带出信息
  - 高阳的抗拒和好奇都要带点公主脾气，而不是平铺讲理
  - 李世民与长孙皇后则要像夫妻、帝后在聊一件看似家事、实则国事的事
- **为什么改**：
  - 用户明确要求：
    - 结构紧凑
    - 行文轻松
    - 吸引读者读下去
    - 避免出现直接“第几章”的描述

## Assumptions & Decisions

- **决策 1**：本次按用户要求，采用“宫中线占主位”的重写方案，不保留现有第十一章作为主体。
- **决策 2**：第十一章中的同辈分流将降为背景结果，不再是核心主线。
- **决策 3**：高阳当前阶段的情绪定位是“先不愿、再起疑、最后决定试探”，不是直接好感。
- **决策 4**：李世民与长孙皇后在本章的功能是打开婚配盘面，不在本章内真正拍板婚配。
- **决策 5**：本轮同步正文与章节小纲，不扩展到其他章节正文与总纲。
- **决策 6**：正文中避免任何直接点“第几章”的元叙事说法，所有承接都通过剧情自然完成。
