# 新增长孙冲与长孙涣角色卡设计计划

## Summary

本轮目标是根据目前剧情的推进和用户的要求，在 `03-人物` 目录下为长孙家族的两位同辈核心角色——**长孙冲**与**长孙涣**新建或完善专属的角色卡。
虽然他们此前分别在《反派二代人物卡》和《主角狐朋狗友人物卡》中有过基础设定，但随着长乐公主感情线的深入、长孙皇后政治态度的倒戈以及主角（房兵）的不断做大，这两兄弟在剧情中的功能和与主角的冲突/合作关系已发生了质变。新建的卡片将更聚焦于他们在**长乐婚配防线**与**长孙家外戚利益**中的具体弧光与定位。

## Current State Analysis

### 1. 现有基础设定
- **长孙冲**：长孙无忌长子，被定位为“伪体面型强敌”。他是长乐公主原本最稳妥的婚配对象，代表着最标准的高门嫡子。他因主角的崛起而感受到了名望与婚配资源的双重威胁（见 `06-反派二代人物卡.md`）。
- **长孙涣**：长孙无忌之子，被定位为“酒局社交型官二代”。他是长安圈的“攒局者”，也是最早察觉主角价值并主动靠拢的狐朋狗友之一（见 `05-主角狐朋狗友人物卡.md`）。

### 2. 剧情演进带来的新变量
- **长孙皇后的倒戈**：在最新的设定中，长孙皇后因被主角救治，已在心理上“撕毁了长孙冲的婚配防线”，甚至为了太子而防备长孙无忌（见 `13-长孙皇后人物卡.md`）。
- **长乐公主的动摇**：长乐正在从“默认既定路线（长孙冲）”向“被主角的降维能力吸引”过渡（见 `04-长乐公主李丽质人物卡.md`）。
- 这意味着，长孙冲将面临**未婚妻与家族靠山（皇后）双双倒向主角**的巨大屈辱与危机；而长孙涣则可能在家族利益与主角阵营之间面临投机的选择。

### 2. 核心内容结构设计

#### 【长孙冲：旧秩序的完美标本与最大输家】
- **基本定位**：
  - 身份：长孙无忌嫡长子，长乐公主原定驸马。
  - 角色关键词：高门楷模、伪体面、极度自尊、被主角碾碎的骄傲。
- **人物弧光与心理演变**：
  - **前期（高高在上的无视）**：视房遗爱为长安的笑话，认为长乐嫁给自己是天经地义、不可动摇的铁律。
  - **中期（体面的打压与破防）**：发现长乐的目光开始偏移，长孙皇后的态度变得暧昧。他不会像市井无赖那样骂街，而是试图通过“清议”、“诗会”或借父辈（长孙无忌）的势来体面地打压主角。
  - **后期（彻底的屈辱与黑化）**：当长孙皇后亲自撕毁婚约、长乐彻底倾心主角后，他引以为傲的“完美人生”被主角降维碾碎。他将成为外戚势力反扑主角的最前锋。
- **剧情功能**：
  - 扮演主角在“长乐公主线”上的最强现实阻力。
  - 代表“旧门阀与外戚利益”的具象化缩影。

#### 【长孙涣：长袖善舞的投机者与家族叛逆】
- **基本定位**：
  - 身份：长孙无忌之子（长孙冲之弟）。
  - 角色关键词：攒局者、情报贩子、极其敏锐的风向标。
- **人物弧光与心理演变**：
  - **前期（凑热闹的狐朋狗友）**：只觉得主角变聪明了、四海楼的酒好喝，利用主角的资源在长安攒局赚面子。
  - **中期（两头下注的试探）**：当他发现兄长长孙冲与主角因为长乐公主发生致命冲突时，他没有盲目站队兄长。他凭借敏锐的政治嗅觉，隐约察觉到主角身上有连父亲（长孙无忌）都看不透的底牌，开始暗中向主角输送长孙家的情报。
  - **后期（彻底的倒戈）**：当外戚势力与主角发生决战时，他选择保全自己，成为主角瓦解长孙家族内部的“带路党”或存续长孙家香火的最后退路。
- **剧情功能**：
  - 充当主角在长安青年圈的“社交与情报接口”。
  - 作为对比组：证明主角的降维打击能力足以撕裂最坚固的顶级门阀内部。

## Assumptions & Decisions
- **决策 1**：将长孙冲的敌意升级。他不仅是“反派二代”，更是因为“夺妻之恨”和“家族资源被抢”而与主角产生不死不休矛盾的核心宿敌。
- **决策 2**：赋予长孙涣更深的“政治投机属性”。他不再只是个单纯的酒肉朋友，而是随着主角做大，演变成门阀内部被主角能力“统战”的典型代表。
- **决策 3**：从原有的 `06-反派二代` 和 `05-狐朋狗友` 卡中提取这两人的核心特质，并在新卡中补全他们与长乐公主、长孙皇后的联动逻辑，使剧情链条完全闭环。
