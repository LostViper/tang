# 新增教坊司双魁设定计划

## Summary

本次任务的核心，是把两位新的风月场女性角色正式钉成稳定设定，并让她们不是“孤立美人”，而是能自然嵌进长安空间、社交网络与后续主角成长线的人物：

- 教坊司花魁：柳凝雪
  - 坊中别称：凝霜娘子
- 醉月坊头牌：苏媚儿
  - 坊中别称：媚骨娘子

两人的方向已经很清楚：

- 柳凝雪偏冷、偏雅、偏清绝
- 苏媚儿偏艳、偏媚、偏外场

因此本轮计划不只是“记录两张美人卡”，而是要完成三件事：

- 把两位人物设定写成可复用的人物卡
- 把她们挂进现有长安“平康坊 / 风月地 / 消息网络”体系
- 把她们纳入第一卷女性角色入口的结构预留中

## Current State Analysis

经过现有材料检索，项目里目前：

- 没有“教坊司 / 花魁 / 醉月坊 / 头牌 / 柳凝雪 / 苏媚儿”的直接设定文本
- 但已经存在非常明确的承接空间：

### 现有最关键的可挂接入口

#### 1. 平康坊已经存在

在 [贞观长安坊市寺观地标.md](file:///Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/01-核心设定/10-贞观长安坊市寺观地标.md#L17-L18) 中，平康坊已经被明确写成：

- 妓馆、举子聚集地
- 长安著名风流薮泽
- 举子、官员、商贾常出入
- 同时具备消息交换和社交结网功能

这意味着新增柳凝雪、苏媚儿时，不需要凭空造新世界，只需要把人物顺势挂到现有平康坊系统里。

#### 2. 第一卷已有“女性角色入口”但缺少风月线

在 [第一卷大纲.md](file:///Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/04-剧情大纲/01-第一卷大纲.md#L164-L172) 里，当前女性角色入口仍主要是：

- 房家与高门社交圈
- 门阀主动试探的贵女入口
- 皇室婚配盘面

也就是说，当前缺少一条“民间风月 / 情报社交 / 才华吸引型”的女性入口。

#### 3. 主角的社交引路人已有天然入口

在 [主角狐朋狗友人物卡.md](file:///Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/03-人物/05-主角狐朋狗友人物卡.md#L102-L130) 中，像长孙涣这类会组局、会通人脉、能带主角进入长安社交场的人已经存在。

这意味着后续若要让主角接触柳凝雪、苏媚儿，路径并不缺，只是角色本身还没建好。

### 当前最需要解决的缺口

#### 1. 两位人物还没有统一角色卡

- 目前用户给出了大量细节
- 但这些细节还没有落进项目的稳定写作模板里
- 如果直接往正文或大纲里塞，会导致后续口径不稳

#### 2. “教坊司”和“醉月坊”还没有和现有平康坊体系接通

- 项目现在有“平康坊”这个总入口
- 但没有更细一级的风月场所命名层
- 若不先补设定，会导致柳凝雪和苏媚儿缺少稳定落点

#### 3. 两人功能若不拆清，后面容易同质化

从你给的设定看，两人最容易写重的地方在于：

- 都很美
- 都是风月场头面人物
- 都会被主角才华吸引

但她们真正应该分开的，是功能：

- 柳凝雪：冷、雅、清绝、疏离、偏诗名与“不可轻犯”感
- 苏媚儿：艳、媚、软、通透、偏场面感与“处处撩人”感

本轮计划必须先把这个差异钉死。

## Assumptions & Decisions

- 本轮任务的目标是“新增设定”，不是立刻扩写她们的正式出场正文
- 两位角色优先视为“平康坊体系下的核心女性入口”，而不是单独脱离长安空间的孤立角色
- “教坊司花魁”与“醉月坊头牌”这两个设定，本轮不强行去做考据纠偏，而是按项目现有架空写法处理：
  - 柳凝雪可以挂在平康坊清牌 / 清名 / 教坊路数的一侧
  - 苏媚儿可以挂在醉月坊这一明确场牌上
- 两人都将作为后续会被主角才华吸引的角色预留，但本轮不把这种“倾慕”直接扩成感情支线
- 这次不主动去重写前四章正文，也不去碰你最近手动调整过的第一章结尾
