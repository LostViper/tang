# 第十七章三档酒命名统一修正计划

## Summary

本轮目标不是重写第十七章整章，而是**修正第十五章与第十七章之间“三档酒命名不一致”的连续性问题**，并把这组酒名的销售定位重新统一清楚。

经过实际检索，当前存在两套口径：

- 第十五章筹备期开出的名字：
  - 照夜白
  - 玉壶春
  - 长安烧
- 第十七章开业章实际写出的名字：
  - 清霜
  - 照春
  - 压城

用户已明确决定：

- **沿用第十五章口径**
- 并补充一条新的定位要求：**第一档限量售卖**

因此本轮计划的核心是：

- 以第十五章为准绳，统一三档酒命名
- 把第十七章中所有不一致命名改回既定名称
- 顺手把“第一档限量售卖”落实为稳定口径
- 检查是否还有相关描述与三档定位冲突

## Current State Analysis

### 1. 当前命名冲突是实打实存在的，而且集中在第十五章与第十七章之间

- 第十五章正文文件：
  - `/Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/11-章节草稿/16-第十五章-筹备开业.md`
- 第十七章正文文件：
  - `/Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/11-章节草稿/18-第十七章-火爆的开业.md`
- 已有事实如下：
  - 第十五章把三档酒名称已经明确写死为：
    - 照夜白
    - 玉壶春
    - 长安烧
  - 第十七章开业章又改成：
    - 清霜
    - 照春
    - 压城
- 这不是单纯措辞差异，而是同一产品体系出现了两套并行命名，读者会直接觉得设定断层。

### 2. 第十五章那套名字已经承担了“筹备期正式定盘”的功能，更适合作为统一标准

- 第十五章里三档酒不是随口一提，而是在“酒水分档 + 瓶器 + 话术 + 档位定位”里一整套定下来的。
- 第十五章既有功能包括：
  - 定名
  - 定档
  - 定售卖场景
  - 定身份感差异
- 因此若继续保留第十五章为准，有天然优势：
  - 不需要回头重做筹备逻辑
  - 读者对“前期定盘 → 开业兑现”会更顺
  - 修改面也更可控

### 3. 第十七章目前不仅名字不同，连档位功能措辞也需要与第十五章重新对齐

- 当前第十七章写法是：
  - 清霜给尝新的人
  - 照春给摆席面的人
  - 压城给压场子的人
- 这组功能虽然方向接近，但和第十五章原设并非一一一致：
  - 第十五章最上头一档“照夜白”强调雅间、门面、身份
  - 中档“玉壶春”强调正席
  - 下档“长安烧”强调尝鲜、走量、外送
- 用户又新增“第一档限量售卖”，说明本轮不只是换名字，还要把顶档定位进一步坐实。

### 4. 当前最稳的改法是“小修第十七章，保留第十五章”为主，而不是两边都大动

- 经过检索，目前明确出现这两套命名的位置，核心集中在：
  - 第十五章筹备段
  - 第十七章开业段及相关后续反应
- 没有发现更大范围的下游扩散。
- 所以最稳路线不是大面积返工，而是：
  - 锁第十五章为标准
  - 修第十七章相关命名与定位
  - 检查小纲与后文是否还引用旧称

## Proposed Changes

### 1. 以第十五章为统一标准，锁定三档酒正式名称

- **统一口径**：
  - 最上档：照夜白
  - 中档：玉壶春
  - 底档：长安烧
- **为什么这么定**：
  - 第十五章已经先完成定盘
  - 用户明确选择沿用第十五章
  - 这套名称已与档位、瓶器、话术绑定

### 2. 修正第十七章中所有三档酒相关命名

- **拟修改文件**：
  - `/Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/11-章节草稿/18-第十七章-火爆的开业.md`
- **怎么改**：
  - 将“清霜”改回“长安烧”或对应底档位置
  - 将“照春”改回“玉壶春”
  - 将“压城”改回“照夜白”
  - 同时把相关客人点单、抱坛、品评、酒力评价等后续称呼全部统一
- **为什么改**：
  - 第十七章是开业兑现章，不能出现和筹备期两套名字并行

### 3. 把第一档“限量售卖”写清楚，并与第十五章原定位并拢

- **拟修改文件**：
  - 第十五章正文
  - 第十七章正文
  - 如有必要，相关小纲补一嘴定位
- **怎么改**：
  - 在第十五章筹备定盘处，补强或明确：
    - 照夜白只进雅间
    - 数量有限
    - 不是人人都能点
  - 在第十七章开业现场兑现时，也让读者看到：
    - 照夜白不是随便敞卖
    - 它的“稀缺 + 身份 + 压场”功能更强
- **为什么改**：
  - 用户已新给出“第一档限量售卖”
  - 这能进一步增强档位层次感，也更符合高端新酒的人设

### 4. 检查是否存在与三档酒相关的小纲或正文残余口径

- **拟检查文件**：

  - `/Users/eric/我带了一整个地球穿越大唐/我带了一整个地球穿越大唐-小说项目/11-章节草稿/16-第十五章-筹备开业.md`

  - 避免正文改完后，大纲或相邻章残留旧称，后面再炸一次连续性问题

## Assumptions & Decisions

- **决策 1**：三档酒正式名称以第十五章已定版本为准：
  - 照夜白
  - 玉壶春
  - 长安烧
- **决策 2**：第十七章中“清霜 / 照春 / 压城”视为需要回收的冲突命名，不保留并行解释。
- **决策 3**：第一档“照夜白”新增明确销售规则：限量售卖。
- **决策 4**：本轮优先修正文连续性，不扩大成整条酒楼线重写。
