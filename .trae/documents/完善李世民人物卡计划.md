# 完善李世民角色设定计划

## Summary

本轮目标是根据用户的最新要求，为李世民新建或完善一份专属的**人物弧光卡**（如果尚未独立建卡，则新建 `14-李世民人物卡.md`）。
核心要点是明确李世民对主角（房兵）从“轻视/利用”到“震撼/依赖”，再到“听闻未来后反思自身、确定太子地位”的完整心理与政治转变。这不仅丰富了李世民的帝王弧光，也为后续主角深度介入储位之争和重塑大唐历史提供了最高层级的逻辑闭环。

## Current State Analysis

### 1. 现有资料中的李世民设定
- **定位**：33岁，英明果断的君主，处于“打天下”向“治天下”过渡期（见《贞观元年朝堂人物群像》）。
- **与子女的关联**：偏爱李泰，导致李承乾安全感缺失，正在无意中破坏储位秩序（见《李承乾李泰人物卡》）。
- **与主角的互动现状**：最初只将主角的“异变”和高阳的婚事视为平衡房家与拉拢功臣的手段。在第17/18章亲自下场看四海楼爆火后，才开始真正认可主角的能力，但仍处于“君王对臣子”的上位者审视阶段。

### 2. 核心写作诉求
- **态度转变链条**：
  1. 一开始没把主角的转变放在心上（高阳婚事只是笼络手段）。
  2. 慢慢感受到主角变化，一步步加深评价。
  3. **关键转折**：主角救治长孙皇后。
  4. **终极震撼与反思**：房父（房玄龄）和主角向他坦白了唐朝后续的发展（历史剧透/推演），这逼迫李世民跳出“局中人”身份，以第三人视角评判自己作为父亲和皇帝的所作所为。
  5. **行动落点**：痛定思痛，下定决心确立太子（李承乾）地位，不再给李泰机会（就藩收束）。

- **决策 1**：将“房父与主角坦白后续唐朝发展”设定为李世民转变的**最高爆点**。这不仅解释了李世民为何突然改变对储位的暧昧态度，也让主角与李世民的关系从“君臣”升华为“共塑大唐的同盟”。
