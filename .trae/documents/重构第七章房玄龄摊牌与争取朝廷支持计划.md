# 重构第七章房玄龄摊牌与争取朝廷支持计划

## Summary

本次目标仍然是完成第七章剧情重构，但计划已根据当前执行进度更新：第七章核心摊牌段、第七章与第八章的小纲方向、第一卷大纲的战略线已经基本改到位；接下来真正需要执行的是**把剩余承接收口**，尤其是第八章正文、标题口径和全局一致性。

这次执行范围继续保持不变：

- 第七章采用“天人传道”外壳，由主角向房玄龄全开直说
- 明说唐朝后续发展、房家原历史下场、房遗爱与高阳结局
- 在战略层面换取房玄龄支持
- 把“先得房玄龄支持，再谋李世民支持”定为前期主线
- 连后续承接一起调整，不把大摊牌做成孤立爆点

- 第一卷章节小纲已改到新方向：
  - 第七章标题与剧情目标已转向“先把未来摊给房玄龄”
  - 第八章已转向“房家表面照旧，内里进入保密与筹备阶段”
- 第一卷大纲的第二幕战略线已改：
  - 明确“仅靠个人具现只是补断点”
  - 明确“必须先拿房玄龄支持，再谋李世民支持”
- 第七章正文的核心摊牌段已实质重写：
  - 主角已用“天人传道”作为叙事外壳
  - 已明说唐史走向、房家结局、房遗爱与高阳恶果
  - 已把“争取李世民支持”落为下一步战略

### 2. 尚未完成部分

- 第八章正文尚未充分体现“房玄龄已知情后，房家内里进入保密与筹备状态”
- 第七章正文标题与文件名仍是旧口径，和章内内容不完全匹配
- 第一卷大纲章节列表中的第七章标题仍可能残留旧口径
- 第七章与第八章之间的知情范围口径仍需统一：
  - 是“父子先结密盟”
  - 还是“房家核心层形成第一层战略同盟”

## Core Judgment

### 1. 当前最大问题已不是“要不要改”

第七章的方向已经改了，真正风险变成：

- 第七章摊得很大
- 第八章接得不够深
- 读者会感觉家里明明听了大事，第二天却像什么都没发生

所以后续执行重点不是继续推翻第七章，而是**把第七章带来的新重心稳稳压进第八章与全局口径里**。

### 2. “天人传道”必须继续保持现实主义落点

后续任何补写都要维持这一尺度：

- 房玄龄不是被神怪说服
- 房玄龄是被信息密度、风险规模、政治窗口说服
- 他可以暂不论真伪，但一定会先控消息、先护家门、先看如何落到陛下层面

## Key Decisions

- **决策 1**：第七章保持“已摊牌”方向，不再回退到“轻试探”。
- **决策 2**：李世民在第七章只作为下一步战略目标，不在本章直接出场解决问题。
- **决策 3**：第八章的改法以“暗线加压”为主，不破坏现有家庭温度与轻松感。
- **决策 4**：知情范围优先采用“房玄龄先单独接密，家内其余人暂不全知”的版本，除非正文连续性要求再放宽。
- **决策 5**：后续所有相关章节都要默认一个前提：主角前期改造大唐，已经从“个人试手”升级为“争取高层支持的系统推进”。

## Risks

- 若第八章承接写得太直，会让节奏过早进入朝堂议政，压坏家常气
- 若第八章承接写得太轻，又会削弱第七章大摊牌的后坐力
- 若知情范围不统一，会造成小纲、大纲、正文三套口径并存
- 若标题不统一，会让读者误判第七章功能

## Verification Steps

- 检查第七章正文标题、第一卷小纲、第一卷大纲是否统一到新口径
- 检查第七章是否完整呈现以下四步：

  - 房玄龄已知情后的行为变化
  - 房家表面照旧、内里加密
  - 对外仍维持“病后开窍”的统一说法
  - 下一步见驾的气息开始埋入
- 检查第一卷大纲与后续章节逻辑是否继续成立：
  - 个人具现只是补断口
  - 房玄龄是第一层政治支点
  - 李世民是下一层战略目标
  - 酒楼线与资源线成为更大布局的前置动作
