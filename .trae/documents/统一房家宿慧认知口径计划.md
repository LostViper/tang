# 统一房家宿慧认知口径计划

## Summary

本次调整的目标，是把第一卷前段关于“落水后变化”的认知口径统一为同一条线：

- 房玄龄并不怀疑主角不是自己的儿子
- 房玄龄的核心判断，是“儿子落水后变聪明了，但落水本身不是孤立事件，背后可能有人或有局”
- “宿慧 / 天外奇景”只能作为内院安人心、解释开窍的柔性说法，不能外传成妖异或夺舍传闻
- 房家其他人只会觉得房遗爱经历生死后性情与脑子都更清明了，不会往“被替换”“换了个人”上怀疑

这次会同步三类内容，避免前后口径打架：

- 第一卷卷级大纲
- 第一卷章节小纲
- 已完成正文中与该设定直接相关的章节

## Current State Analysis

### 已确认的当前冲突

经过对现有大纲与正文的只读核对，当前冲突主要集中在两类：

1. **身份怀疑写得过重**
   - 现稿中多处把房玄龄的怀疑推进到“是不是自己的儿子”“是不是醒成了别的东西”
   - 这与本次新设定冲突，因为新设定要求房玄龄默认“这就是他儿子”，只是警惕“为什么会忽然开窍、为什么会落水、背后是谁动的手”

2. **内外口径写得过险**
   - 现稿中“宿慧”“撞邪”“回魂”“换了个人”“换了芯子”等表达分布较散
   - 有些地方已写成全家或相府都会往妖异、替换方向联想
   - 这会把后续房家认知战写偏，削弱“调查落水真因”和“高门家内重新估价”的现实张力

### 目前最需要纠偏的具体问题

#### 1. 第一卷大纲

- 第二幕与第三幕中，把房家核心矛盾写成了“附体 / 妖异 / 换了个人 / 回魂”这一类高烈度判断
- 需要改回“开窍之后的重新估价 + 落水真因调查 + 房家内部保护与观察并行”

#### 2. 第一卷章节小纲

- 第 2 章写到“房玄龄是否怀疑不是亲儿子”
- 第 3 章写到银铃儿几乎确认“换了芯子”
- 第 7 章当前写成“回魂、开窍还是撞邪”的核心盘问
- 第 10、11、15 章也延续了“宿慧外传”“像换了个人”“撞邪流言”的旧口径
- 这些都需要统一改成更稳、更现实的表达

#### 3. 已完成正文

- 第一章收束段把相府猜测写成“回魂 / 开窍 / 撞邪”
- 第二章存在“是不是你儿子”“妖物”等直白表达
- 第三章把银铃儿的敏感点写成“不能太像忽然换了个人”
- 第七章把父子交锋核心写成“怕他醒成别的东西”“若答我其实不是你儿子”等
- 这些地方都要改回“仍是房遗爱，只是变得更聪明、更稳、更会想事”

## Proposed Changes

### 一、统一卷级叙事方向

目标文件：

- `我带了一整个地球穿越大唐-小说项目/04-剧情大纲/01-第一卷大纲.md`

调整方式：

- 把“换魂 / 回魂 / 妖异 / 被替换”的强烈表述，统一收束成“死里逃生后性情与脑子都变了”
- 明确房玄龄的主关注点是：
  - 落水是不是被人做局
  - 主角醒后为何突然变聪明
  - 这些变化对房家是机会还是风险
- 保留“宿慧”作为内院安人心的解释壳，但不再把它写成外部舆论主口径
- 把第二幕到第三幕的核心压力，重新落到：
  - 房家内部重新估价
  - 相府上下观察
  - 落水真因调查
  - 主角如何在“变聪明但不能太妖”的边界上站稳

预期效果：

- 卷级大纲与正文逻辑重新对齐
- 第一卷前段从“灵异怀疑”回到“权门调查 + 家内认知战”

### 二、重写第一卷章节小纲中的认知口径

目标文件：

- `我带了一整个地球穿越大唐-小说项目/04-剧情大纲/13-第一卷章节小纲.md`

具体调整：

#### 第 1-3 章

- 第 1 章：
  - 保留“天外奇景”的震感
  - 删弱“回魂 / 撞邪”联想
  - 改成“人醒了、见识怪了、说话比从前清楚了”
- 第 2 章：
  - 房玄龄不再怀疑“是不是自己的儿子”
  - 改成“要先查落水经过，再判断这场变化到底怎么来的”
  - 房母也不再把矛盾顶到“你把孩子看成妖物”，而是顶到“你别把刚醒的孩子当案子查得太狠”
- 第 3 章：
  - 银铃儿保留敏感和异样感
  - 但她的判断要改成“二郎像突然长了脑子、说话稳了”，而不是“换了个人”

#### 第 7 章

- 把章节目标从“骗过一个怀疑你是不是原装的房玄龄”
  调整为
  “先稳住房玄龄对落水事件的追查和对儿子突然开窍的警惕”
- 把场景推进改成：
  - 房玄龄借问病情、落水、旧事，确认儿子神智是否稳定
  - 主角把现代见闻收束为“生死之间见过许多远景，所以心智像被撞开了些”
  - 房玄龄认可“宿慧只能在内院当说法”，但真正要查的是落水真因和儿子变化背后的触发点
- 把章末钩子改成：
  - 房玄龄怀疑“此事背后有人”
  - 而不是怀疑“眼前这个人不是房遗爱”

#### 第 10、11、15 章

- 第 10 章：
  - 保留房母护短与金铃儿传话
  - 但“宿慧”改为内院柔性说法，不扩成对外流言
- 第 11 章：
  - 银铃儿的观察点改成生活习惯和说话方式变稳，不再是“像换了个人”
- 第 15 章：
  - 同辈圈流言改为“房遗爱回了一趟鬼门关后突然开窍”
  - 不再写成“回魂 / 换魂 / 撞邪”并行主流判断

预期效果：

- 章纲后续扩写时，不会再把房家前段主压力写歪
- 第 7 章到第 15 章的口径将与新设定完整闭环

### 三、回修已完成正文中的关键台词与叙事判断

目标文件：

- `我带了一整个地球穿越大唐-小说项目/11-章节草稿/02-第一章-一局文明梦，醒来已是房遗爱.md`
- `我带了一整个地球穿越大唐-小说项目/11-章节草稿/03-第二章-房母先护，房相先压.md`
- `我带了一整个地球穿越大唐-小说项目/11-章节草稿/04-第三章-金铃儿与银铃儿守着二郎.md`
- `我带了一整个地球穿越大唐-小说项目/11-章节草稿/08-第七章-先骗过房玄龄.md`

具体处理：

#### 第一章

- 收束结尾处“回魂 / 撞邪”一类总括
- 改成相府上下只觉得：
  - 二郎活过来了
  - 说出来的话怪
  - 像忽然开了窍
  - 但没人敢立刻把事情往灵异替换上坐实

#### 第二章

- 删除或改写“是不是你儿子”
- 删除或改写“妖物”指向
- 保留房玄龄的审视感，但审视对象从“身份真假”改为：
  - 精神是否稳定
  - 落水前见过谁
  - 为什么一醒来就变了
- 保留房母与房玄龄的冲突，但冲突重心变成“查得太急 / 吓着孩子”而非“你把他当妖物”

#### 第三章

- 把“不能太像忽然换了个人”改成“不能跟从前差得太扎眼”
- 把内院统一口径改得更稳：
  - 先说病后神思未稳
  - 再说生死关头像是把人撞明白了
  - 最多说一句祖宗保佑，不把“宿慧”拔得太妖

#### 第七章

- 去掉主角和叙述中“不是你儿子”“醒成了别的东西”这类内核判断
- 房玄龄的问法改为：
  - 你为什么突然会想事了
  - 你病中看见的东西到底该怎么解释
  - 你落水前后到底有没有异常
  - 你现在记事和行事为什么与从前不同
- 保留“宿慧只能内院流通、外传不妥”的判断
- 但把这一判断的理由从“怕别人觉得撞邪”偏向：
  - 高门是非多
  - 异言外传会惹来无谓猜测
  - 房家不该自己把话柄递出去
- 章末钩子改成：
  - 房玄龄确认这场变化不简单
  - 并准备继续追查落水真因与变化来源
  - 而不是确认“儿子不对劲到像不是原装”

预期效果：

- 已完成章节与新设定直接对齐
- 房玄龄的人设更像现实权相，而不是先走到玄怪判断
- 房家其他成员的情绪会更自然地落在“惊、喜、护、观察”上

### 四、控制改动边界，避免误伤已稳定设定

不改内容：

- 第五章与第六章蓝星治理主线
- 房家家庭温情底色
- 主角“先藏七分”的生存逻辑
- 宿慧壳作为阶段性保护说法的存在本身

只改内容：

- 对“宿慧”的传播范围
- 对“落水后变化”的解释力度
- 对房玄龄怀疑对象的重心
- 对银铃儿、房母、同辈圈的体感表述

## Assumptions & Decisions

### 已锁定决策

- 本次同步范围包含：
  - 第一卷大纲
  - 第一卷章节小纲
  - 已完成章节
- 不只修你点名的小纲文件，也一起同步卷级大纲，避免文档层口径打架

### 执行原则

- 原则一：房玄龄始终认定眼前人就是房遗爱
- 原则二：房玄龄真正要查的是落水原因与变化成因
- 原则三：房家其他人只会觉得“二郎开窍了”，不会怀疑“被替换了”
- 原则四：宿慧只能当内院缓冲说法，不能发展成外部主流舆论
- 原则五：把“玄怪感”收弱，把“现实权门调查感”提强

### 风格取向

- 继续保留现有轻松幽默底色
- 但幽默用于缓压，不用于把认知冲突写成鬼怪误会
- 房玄龄继续有压迫感，只是压迫来自审人、控局、查事，不来自怀疑儿子被夺舍

## Verification Steps

执行后按以下顺序复核：

1. **卷级大纲复核**
   - 确认第二幕、第三幕不再把“换魂 / 妖异 / 不是亲儿子”当核心矛盾
   - 确认卷级冲突已改成“开窍后的重新估价 + 落水真因调查”

2. **章节小纲复核**
   - 确认第 2、3、7、10、11、15 章已全部改口
   - 确认没有残留“换了芯子 / 不是他儿子 / 醒成别的东西 / 妖物”等冲突表述

3. **已完成章节复核**
   - 逐章检查第 1、2、3、7 章
   - 确认房玄龄不再直接怀疑身份替换
   - 确认银铃儿与房母不再把变化理解成“换了个人”
   - 确认第七章章末钩子落在“继续查落水真因”而非“身份真假”

4. **跨文件一致性复核**
   - 用关键词回查：
     - “不是你儿子”
     - “妖物”
     - “换了个人”
     - “换了芯子”
     - “醒成了别的东西”
   - 确认这些词要么被删除，要么被改写成符合新设定的表达

5. **连贯性复核**
   - 确认第 7 章与第 8 章承接仍顺
   - 确认第 9 章“相府重新估价二郎”仍成立
   - 确认后续“房家观察主角”“房玄龄继续试探”“主角先藏七分”三条线依然能顺着往下写
